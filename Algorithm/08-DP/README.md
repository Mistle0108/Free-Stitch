# 08. DP

## 변경 목적
- 동적 계획법을 점화식 암기가 아니라 상태 설계 훈련으로 정리한다.
- 작은 문제의 답을 이용해 큰 문제를 푸는 사고를 체계화한다.

## 이 폴더에서 다룰 내용
- DP 기초
- 1차원 DP
- 2차원 DP
- 경로 복원
- 배낭 문제
- LIS / LCS
- 트리 DP, 비트마스크 DP 기초

## 왜 중요한가
- 코딩 테스트에서 난도 체감이 크고, 실력 차이가 잘 드러나는 영역이다.
- 한 번 이해하면 다양한 문제에 재사용할 수 있다.

## 학습 목표
- 상태, 점화식, 초기값을 직접 설계할 수 있다.
- 탑다운과 바텀업의 차이를 이해한다.
- DP와 완전탐색의 차이를 중복 상태 관점에서 설명할 수 있다.

## 세부 분류
### 1. DP 기본
- 상태 정의
- 점화식 세우기
- 초기값 설정

### 2. 대표 유형
- 계단 오르기
- 연속합
- 배낭
- LIS
- LCS
- 경로 수 세기

### 3. 심화 방향
- 경로 복원
- 트리 DP
- 비트마스크 DP
- 최적화 기법

## 자주 하는 실수
- 상태를 애매하게 잡아 점화식이 꼬이는 실수
- 초기값을 잘못 둬서 전체 결과가 틀리는 실수
- 점화식은 맞는데 인덱스 처리에서 틀리는 실수
- 중복 상태를 인식하지 못하고 DFS로만 푸는 실수

## 대표 문제 유형
- 최소/최대 비용
- 경우의 수 세기
- 부분 수열 최적화
- 경로 추적
- 선택/비선택 문제

## 추천 정리 방식
```md
## 문제 이름

### 상태 정의
- dp[i] 또는 dp[i][j]가 의미하는 것

### 점화식
- 어떤 이전 상태에서 오는지

### 초기값
- 시작 조건

### 순회 순서
- 왜 그 순서로 계산하는지

### 주의점
- 인덱스, 음수, 범위, 오버플로우
```

## 체크리스트
- [ ] 상태를 한 문장으로 정의할 수 있다.
- [ ] 점화식을 설명할 수 있다.
- [ ] 초기값이 왜 그렇게 되는지 설명할 수 있다.
- [ ] 탑다운/바텀업 중 무엇이 더 자연스러운지 판단할 수 있다.
- [ ] 대표 유형별 DP 패턴을 최소 한 번씩 정리했다.

## 추천 학습 순서
1. 1차원 DP 기초
2. 2차원 DP
3. LIS / LCS
4. 배낭
5. 경로 복원
6. 트리 DP / 비트마스크 DP 기초

## 복습 기준
- 새로운 DP 문제를 봤을 때 상태 정의를 먼저 시도할 수 있으면 좋은 흐름이다.
