# 10. 트리

## 변경 목적
- 그래프 중에서도 트리를 별도 주제로 정리해 구조적 사고를 강화한다.
- 부모-자식 관계, 서브트리, 경로 개념을 중심으로 문제를 정리한다.

## 이 폴더에서 다룰 내용
- 트리 기본 개념
- 순회
- 이진 트리 / BST 기초
- 서브트리
- LCA 기초
- 트리 쿼리 입문

## 왜 중요한가
- 트리는 그래프보다 구조가 명확해서, DFS/DP/쿼리 기법과 결합하기 좋다.
- 면접과 코테 모두에서 자주 등장하는 핵심 주제다.

## 학습 목표
- 트리의 특징을 일반 그래프와 비교해 설명할 수 있다.
- 전위/중위/후위 순회를 구현할 수 있다.
- 루트 기준으로 부모/깊이/서브트리를 다룰 수 있다.

## 세부 분류
### 1. 트리 기본
- 노드, 간선, 루트, 리프
- 부모/자식/깊이
- 트리의 성질

### 2. 순회
- 전위 순회
- 중위 순회
- 후위 순회
- 레벨 순회

### 3. 심화
- 서브트리 크기
- 트리 경로
- 최소 공통 조상
- 트리 DP 연결

## 자주 하는 실수
- 루트 기준을 정하지 않고 문제를 보다가 구현이 꼬이는 실수
- 부모 방문을 막지 않아 무한 재귀에 빠지는 실수
- 트리 순회 순서를 혼동하는 실수
- LCA 전처리 개념을 막연하게 외우는 실수

## 대표 문제 유형
- 트리 순회 결과 출력
- 부모 찾기
- 서브트리 크기 계산
- 공통 조상 찾기
- 트리 경로 거리 계산

## 정리 포인트
- 루트가 주어지지 않으면 어떻게 잡을지 기록한다.
- DFS로 해결되는지, 별도 전처리가 필요한지 구분한다.
- 서브트리 문제는 "현재 노드 기준" 의미를 항상 적는다.

## 체크리스트
- [ ] 트리의 정의와 특징을 설명할 수 있다.
- [ ] 기본 순회를 구현할 수 있다.
- [ ] 루트 기반 DFS로 부모/깊이를 구할 수 있다.
- [ ] LCA가 필요한 상황을 말할 수 있다.
- [ ] 트리 문제를 그래프와 구분해서 접근할 수 있다.
