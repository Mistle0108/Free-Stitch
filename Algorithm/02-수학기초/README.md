# 02. 수학기초

## 변경 목적
- 코딩 테스트에서 반복적으로 등장하는 기본 수학 개념을 정리한다.
- 복잡한 수학 이론보다 구현 가능한 수준의 정수론 감각을 먼저 확보한다.

## 이 폴더에서 다룰 내용
- 약수와 배수
- 소수 판별과 에라토스테네스의 체
- 최대공약수와 최소공배수
- 나머지 연산
- 조합 기초
- 경우의 수 계산 감각

## 왜 중요한가
- 수학 문제 전용이 아니어도, 배열/문자열/그래프 문제에서 수식 정리가 필요할 때가 많다.
- 소수, gcd, 조합은 알고리즘 문제에서 자주 재사용되는 기초 도구다.

## 학습 목표
- 약수/배수/소수 관련 기본 문제를 직접 구현할 수 있다.
- gcd/lcm를 이용해 문제를 단순화할 수 있다.
- 조합을 식으로만 이해하지 않고 코드로 다룰 수 있다.
- 나머지 연산이 필요한 이유를 설명할 수 있다.

## 세부 분류
### 1. 약수와 배수
- 약수 구하기
- 배수 판정
- 공약수/공배수 개념

### 2. 소수
- 소수 판별
- 체를 이용한 전처리
- 소인수분해 기초

### 3. gcd / lcm
- 유클리드 호제법
- 최소공배수 계산
- 비율/분수 단순화

### 4. 조합론 기초
- 순열과 조합 차이
- 조합 공식 이해
- 큰 수 처리 필요성 인지

## 자주 하는 실수
- 소수 판별 시 `sqrt(n)`까지만 보면 된다는 점을 놓치는 실수
- gcd를 직접 반복문으로 비효율적으로 구하는 실수
- 조합 계산에서 오버플로우를 고려하지 않는 실수
- 나머지 연산 분배법칙을 잘못 적용하는 실수

## 대표 문제 유형
- 소수 개수 세기
- 약수 합 구하기
- gcd/lcm 계산
- 조합 값 구하기
- 나머지 규칙을 이용한 계산 문제
