# 14. 기하

## 변경 목적
- 좌표, 선분, 각도, 면적 등 기하 문제의 기초 개념을 정리한다.
- 계산 실수와 부동소수점 이슈를 줄이는 방향으로 공부한다.

## 이 폴더에서 다룰 내용
- 점과 벡터 기초
- 직사각형/삼각형
- 거리 계산
- CCW
- 선분 교차
- 볼록 껍질 기초

## 왜 중요한가
- 출제 빈도는 상대적으로 낮지만, 나오면 별도 개념 없이는 접근이 어렵다.
- 좌표 기반 문제를 수식과 조건으로 정확히 처리하는 훈련이 된다.

## 학습 목표
- 점, 선분, 방향 관계를 수식으로 표현할 수 있다.
- CCW의 의미를 이해하고 활용할 수 있다.
- 선분 교차 조건을 코드로 옮길 수 있다.

## 세부 분류
### 1. 기하 기초
- 점 좌표
- 거리와 기울기
- 면적 계산

### 2. 방향 판정
- 외적
- CCW
- 회전 방향 판단

### 3. 선분/도형
- 선분 교차
- 다각형 기초
- 볼록 껍질

## 자주 하는 실수
- 정수 계산으로 가능한 문제를 부동소수점으로 풀어 오차가 생기는 실수
- 기하 공식을 외우기만 하고 의미를 이해하지 못하는 실수
- 경계 조건(끝점 포함 여부)을 놓치는 실수

## 대표 문제 유형
- 삼각형/사각형 판별
- 거리 계산
- 점의 위치 관계
- 선분 교차 여부
- 볼록 껍질

## 정리 포인트
- 공식만 적지 말고, 그림이나 의미를 함께 적는다.
- 경계 조건 포함 여부를 반드시 적는다.
- 정수 계산 우선, 실수 계산은 필요한 경우에만 사용한다.

## 체크리스트
- [ ] CCW를 설명하고 구현할 수 있다.
- [ ] 선분 교차의 핵심 조건을 말할 수 있다.
- [ ] 부동소수점 오차 위험을 알고 있다.
- [ ] 좌표 문제를 그림으로 먼저 정리하는 습관이 있다.

## 복습 기준
- 기하 문제는 공식 암기보다 조건 해석이 중요하다. 핵심 판정식을 스스로 유도하거나 설명할 수 있으면 좋다.
