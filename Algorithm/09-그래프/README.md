# 09. 그래프

## 변경 목적
- 그래프 문제의 기본 개념과 순회 패턴을 정리한다.
- 연결 관계를 다루는 문제를 DFS/BFS/최단경로/유니온파인드 관점으로 분해할 수 있게 만든다.

## 이 폴더에서 다룰 내용
- 그래프 표현
- DFS
- BFS
- 위상 정렬
- 최단 경로
- 유니온 파인드
- 최소 신장 트리

## 왜 중요한가
- 그래프는 코딩 테스트 핵심 축 중 하나다.
- 문제 문장에 그래프라는 단어가 없더라도 관계/연결/전파/순서가 나오면 그래프일 가능성이 높다.

## 학습 목표
- 문제를 그래프로 모델링할 수 있다.
- 인접 리스트와 인접 행렬의 차이를 이해한다.
- DFS와 BFS를 상황에 맞게 선택할 수 있다.
- 위상 정렬, 다익스트라, 유니온파인드, MST의 기본 목적을 설명할 수 있다.

## 세부 분류
### 1. 그래프와 순회
- 인접 리스트
- 방문 배열
- 연결 요소
- 경로 탐색

### 2. 위상 정렬
- 선행 관계
- DAG
- 진입 차수

### 3. 최단 경로
- BFS 기반 최단 거리
- 다익스트라
- 플로이드-워셜 기초

### 4. 유니온 파인드 / MST
- 집합 합치기
- 사이클 판별
- 크루스칼

## 자주 하는 실수
- 그래프 방향성 여부를 놓치는 실수
- 방문 체크 시점을 잘못 잡는 실수
- BFS 최단거리와 가중치 그래프를 혼동하는 실수
- 유니온파인드에서 경로 압축/루트 비교를 잘못하는 실수

## 대표 문제 유형
- 연결 요소 개수
- 미로 탐색
- 순서 정하기
- 최단 거리 계산
- 친구 관계/집합 합치기
- 네트워크 연결 비용

## 추천 정리 방식
```md
## 알고리즘 이름

### 언제 쓰는가
- 문제 신호

### 핵심 아이디어
- 한 줄 설명

### 구현 포인트
- 자료구조, 방문 처리, 초기값

### 주의점
- 틀리기 쉬운 부분
```

## 체크리스트
- [ ] 문제를 노드와 간선으로 바꿔 설명할 수 있다.
- [ ] DFS/BFS 차이를 말할 수 있다.
- [ ] 위상 정렬이 필요한 상황을 이해한다.
- [ ] 다익스트라 적용 조건을 안다.
- [ ] 유니온파인드를 직접 구현할 수 있다.
- [ ] MST의 목적을 설명할 수 있다.

5. 유니온 파인드
6. MST

## 복습 기준
- 연결/전파/선후관계/최단거리/집합합치기 문제를 보면 그래프 하위 주제로 바로 연결할 수 있어야 한다.
